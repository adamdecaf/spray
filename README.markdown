# Documentation

**Spray has been officially been declared End of Life by the akka team. The official recommendation is to migrate towards akka-http.** Please see our [migration guide](http://doc.akka.io/docs/akka-http/current/scala/http/migration-guide/migration-from-spray.html) for moving apps over to use akka-http.
